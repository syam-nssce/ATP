Course Overview

This course introduces students to computer programming and computational problem solving, with a focus on real-world engineering and scientific problems. The course uses the Java programming language and emphasizes object-oriented programming.

What You'll Learn

  • Fundamentals of Java programming
  • Object-oriented design principles
  • Real-world problem solving using computation
  • Modeling, simulation, and GUI development
